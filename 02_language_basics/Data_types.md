# Object types or Data types

- Number : 1234, 3.414, 3+4j, 0b111, Decimal(), Fraction()
- String : 'spam, "Bob's", b'a\x01c', u'sp\xc4m'
- List : [1 , [2, 'three'], 4, 5], list(range(10))   
-> indexing starts from 0
- Tuple : (1, 2, 3), (1, 'spam', 4.5), tuple('spam')
- Dictionary : {'food': 'spam', 'quantity': 4, 'color': 'pink'}, dict(name='Bob', age=40)
- Set : {1, 2, 3}, set([1, 2, 2, 3, 3, 3])
- File : open('data.txt'), sys.stdin, sys.stdout, open(r'C:\ham.bin', 'rb')

- Boolean : True, False (subclass of integers
- NoneType : None -> empty value
 - Functions : def func(): pass, lambda x: x+1
- Classes/Types : class MyClass: pass, type('MyClass', (), {})
- Modules : import math, import sys

Advance: Decorators, Generators, Iterators, Meta programming


- strings are treated as arrays of characters
